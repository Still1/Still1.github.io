## 常用语法

<!--more-->

### 标题

```
#
```

### 无序号列表

```
*
```

### 有序号列表

```
1.
```

### checkbox未打勾

```
* [ ]
```

### checkbox已打勾

```
* [x]
```

### 代码行

```
``
```

### 代码块

````
```java
```
````

### 斜体

```
* *
```

### 加粗

```
** **
```

### 粗斜体

```
*** ***
```

### 超链接

```
[text](url)
```

### 图片

```
![text](url)
```

### 表格

```
|标题1|标题2|标题3|
|:---|:---:|---:|
|内容1|内容2|内容3|
```

### 引用

```
>
```

### Emoji

```
:code:
```

Emoji编码列表：https://github.com/caiyongji/emoji-list

### 分隔线

```
---
```

### 删除线

```
~~ ~~
```
