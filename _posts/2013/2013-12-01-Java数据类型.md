## Java数据类型以及区分

Java虚拟机规范，数据类型分为

* 基本类型 primitive type
* 引用类型 reference type

能够赋值为`null`的，则是引用类型，否则为基本类型

## Java基本类型及其长度

| 基本类型 | 长度                          |
| -------- | ----------------------------- |
| byte     | 1 Byte                        |
| short    | 2 Byte                        |
| int      | 4 Byte                        |
| long     | 8 Byte                        |
| char     | 2 Byte                        |
| float    | 4 Byte                        |
| double   | 8 Byte                        |
| boolean  | 4 Byte，boolean数组中为1 Byte |

## Java引用类型及其长度

Java引用类型分为

* 类引用
* 接口引用
* 数组引用

一般来说，引用类型变量的长度与JVM是32位还是64位有关，32位的JVM长度为4 Byte，64位的JVM长度为8 Byte。
