## 配置

<!--more-->

`vue.config.js`增加以下配置

```js
devServer: {
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      pathRewrite: {
        '^/api': ''
      }
    }
  }
}
```



假设Vue-CLI运行在`http://localhost:8081`，访问`http://localhost:8081/api/test`将代理到`http://localhost:8080/test`
