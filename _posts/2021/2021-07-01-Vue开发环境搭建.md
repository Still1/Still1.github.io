## 安装Node.js

安装完成后，查看版本确认安装成功

<!--more-->

```shell
node -v
```

## 配置npm镜像

```shell
npm config set registry https://registry.npm.taobao.org
```

## 安装Vue CLI

```shell
npm install -g @vue/cli
```

## 创建项目

```shell
vue create language-trainer-vue
```

## 启动项目

```shell
npm run serve
```
