## 生成项目模板

<!--more-->

简单webapp项目模板， 选择maven-archetype-webapp，搜索maven:webapp

简单项目模板， 选择maven-archetype-quickstart，搜索maven:quickstart

```mvn archetype:generate```

## 构建项目

```
mvn clean test site
```

```
mvn clean package site
```

## 生成测试覆盖率报告

```
mvn cobertura:cobertura
```
