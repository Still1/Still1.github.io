## 日期处理相关的类的含义

<!--more-->

| 类名            | 含义                   |
| --------------- | ---------------------- |
| `LocalDate`     | 不含时区信息的日期     |
| `LocalTime`     | 不含时区信息的时间     |
| `LocalDateTime` | 不含时区信息的日期时间 |
| `Instant`       | 不含时区信息的一瞬间   |
| `ZoneId`        | 一个时区               |
| `ZonedDateTime` | 含时区信息的日期时间   |
| `Clock`         |                        |
| `Duration`      |                        |
